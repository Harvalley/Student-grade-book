# Student-grade-book
This project is a simple student grade book system built with Python. It allows users to store and manage student names along with their scores.
How It Works
The user enters student names
The user inputs scores for each student
The program stores the data
It can display:
All students and their scores
Average score
Highest and lowest scores

Features
Add student names and grades
Store data using lists or dictionaries
Calculate average score 📊
Find highest and lowest grades
Display all records
